## 📂 **GITHUB PAGES SETUP**

### **Step 1: Create Repository**
1. Go to GitHub.com
2. Click **"New repository"**
3. Name it: `help-center` (or any name)
4. Make it **Public** (required for free GitHub Pages)
5. Click **"Create repository"**

### **Step 2: Upload Files**
1. Click **"uploading an existing file"**
2. Drag `index.html` into the upload area
3. Commit: "Initial commit - H.E.L.P. Center"

### **Step 3: Enable GitHub Pages**
1. Go to repository **Settings**
2. Scroll to **Pages** (left sidebar)
3. Under **Source**, select **"main"** branch
4. Click **Save**
5. Wait 1-2 minutes for deployment

### **Step 4: Access Your Site**
Visit: `https://YOUR-USERNAME.github.io/help-center/`

## 🛠️ **TROUBLESHOOTING**

### **Page not loading?**
- Clear browser cache (Ctrl + Shift + Delete)
- Try incognito/private mode
- Check GitHub Pages deployment status

### **Buttons not working?**
- Check browser console (F12 → Console)
- Look for JavaScript errors
- Make sure JavaScript is enabled

### **Notability badge not showing?**
- Refresh page (F5)
- Check if topbar is visible
- Zoom out if needed (Ctrl + -)

### **Import button not working?**
- Click the button
- File browser should open
- Select a .json file
- Confirm import
